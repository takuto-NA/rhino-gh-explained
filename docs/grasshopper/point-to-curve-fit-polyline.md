# Fit Curve（近似）とPolyline（折れ線）

このページは、`point-to-curve.md` の詳細版です（ノイズ・角の保持の判断に集中します）。

## Fit Curve（近似）：誤差許容内で点列に近似する

**Q: Fit Curveコンポーネントの基本的な動作は？**

A: `Fit Curve` 系は、点列を **誤差（許容値）以内**で近似する曲線を作ります。

**Q: Fit Curveの保証事項は？**

A: すべての点を通過させるのではなく、許容誤差内で近似することを保証します。

**Q: 適用場面は？**

A: 点列がノイズを含む（測定/スキャン/サンプル等）場合に適用します。

**Q: 調整の要点は？**

A: 許容誤差を小さくすると補間に近づき、大きくすると滑らかになる一方、形状差も大きくなります。

## Polyline（折れ線）：角を保持しつつ点列を接続する

**Q: 角を保持したい場合は？**

A: `Polyline` を第一候補とするのが安全です。

**Q: Polylineの保証事項は？**

A: 点を直線で結ぶ（挙動が明確）ことを保証します。

**Q: 後処理は？**

A: 必要に応じて局所的に `Fillet` / `Blend` / `Chamfer` を適用します。

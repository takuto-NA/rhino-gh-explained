# 可変厚み（1mmベース＋局所補強）を破綻させない手順

アニマトロニクス外装は「基本1mm」でも、首周り・コネクタ周り・分割フランジなどで**局所的に厚く**する必要があります。  
ここでのポイントは、**最初から可変厚みにしない**ことです。

## 結論：安全な順序

1. **均一厚みで“閉じた殻”を作る**（まず1mm相当）\n2. **前後分割/首開口/穴あけの設計を入れる**（位置を確定できる基準を持つ）\n3. **局所補強を“足す”**（ボス/フランジ/リブ/当たり面）\n4. 最後に **出力検証**（スライサーで壁数・薄肉・サポートを確認）

## “均一厚み”の作り方（形状の種類で分岐）

どの表現（Mesh/SubD/Brep）で進めるかで手段が変わります。

- **Brep（NURBSソリッド）**: `Shell` が最短\n- **開いた面（サーフェス）**: `OffsetSrf`（必要ならSolid=Yes）\n- **SubD/メッシュ**: まずは“殻を作れる表現”に寄せる\n+  - どのみち最終出力はSTL/3MFなので、\n+    「壊れない殻」になっているか（穴/非多様体/自交差が無いか）が最重要

## 可変厚みを“足す”方法（FDM向け）

局所補強は、外装そのものを膨らませるより **別パーツとして追加**する方が安定します。

- **フランジ（合わせ面）**: 前後分割の合わせ面を作る（面で組む）\n- **リブ**: 広い薄板がたわむ/割れる箇所に入れる\n- **ボス**: ねじ止め、機構固定、嵌合の基礎になる\n- **当たり面**: 機構CADと接触する箇所に平面/円筒などの“基準面”を作る

## 失敗の典型と対策

- **強曲率部で自己交差**\n  - 対策: その領域を先に分割し、局所ごとに処理する\n- **薄肉のまま大開口を作って割れる**（首周り）\n  - 対策: 開口縁にフランジ/リブを前提で設計する\n- **可変厚みを先に入れて、後段の穴あけ/分割で破綻**\n  - 対策: 先に均一殻→設計（分割/穴）→最後に補強

## 次にやること

次章では、**前後2分割**、**首の大開口**、**目などの後加工穴**を\n+「後で位置を変えられる基準の持ち方」込みで整理します。

# 分割（前後2分割）・首開口・後加工穴（目など）

この章は「外装をあとから組む／内部機構を入れる」前提で、\n+**分割**と**大開口**と**穴あけ**を“後で確定できる”ように設計する考え方をまとめます。

## 前後2分割：分割面の決め方

分割面は、見た目だけで決めると後で詰みやすいです。優先順位は以下です。

- **組付けしやすい**（手が入る、ネジが回る、位置決めできる）\n- **プリントしやすい**（サポートが減る、歪みにくい）\n- **見た目が自然**（ラインが目立ちにくい）

### 合わせ面（フランジ）を必ず作る

有機形状の断面同士を直接合わせるのは不安定です。

- 周方向に **フランジ**を作る（接着/ねじ/嵌合/マグネットの土台）\n- フランジはできるだけ **平面または一定断面**を確保する\n- 位置決め要素（ダボ、段差、キー）を入れると組付けが楽

## 首の大開口：空洞化と強度の両立

首開口は構造上の弱点になりやすいです。

- 開口縁に **厚みと幅を確保**（縁が欠ける/割れるのを防ぐ）\n- 必要なら **リング状の補強**や、内側に **リブ**を入れる\n- 分割と首開口が近い場合は、フランジと補強が干渉しないように先に設計する

## 目などの後加工穴：基準を持って“後で確定”する

「目は後で穴を開ける」場合に重要なのは、穴そのものより **穴の基準**です。

### 基準の作り方（おすすめ）

- **基準平面**: 正中面、目の高さ面、顔の前後基準面\n- **基準点**: 目頭/目尻、瞳中心、鼻先、口角など\n- **基準線**: 正中ライン、目のライン（左右）\n
穴あけは、\n+「基準 → 寸法（半径/楕円） → 位置決め」\n+の順にすると、変更が入っても追従しやすいです。

### 穴の縁（エッジ）を“設計要素”にする

穴は開けた瞬間に局所応力が集中します。\n+FDMでは特に割れやすいので、\n+穴縁周りに **厚み**や **当たり面**を前提として残す設計が安全です。

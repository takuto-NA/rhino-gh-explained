# 分割（前後2分割）・首開口・後加工穴（目など）

この章は「外装をあとから組む／内部機構を入れる」前提で、**分割**と**大開口**と**穴あけ**を“後で確定できる”ように設計する考え方をまとめます。

## 前後2分割：分割面の決め方

分割面は、見た目だけで決めると後で詰みやすいです。優先順位は以下です。

- **組付けしやすい**（手が入る、ネジが回る、位置決めできる）
- **プリントしやすい**（サポートが減る、歪みにくい）
- **見た目が自然**（ラインが目立ちにくい）

### 合わせ面（フランジ）を必ず作る

有機形状の断面同士を直接合わせるのは不安定です。

- 周方向に **フランジ**を作る（接着/ねじ/嵌合/マグネットの土台）
- フランジはできるだけ **平面または一定断面**を確保する
- 位置決め要素（ダボ、段差、キー）を入れると組付けが楽

## Rhinoコマンド：前後2分割とフランジ作成（具体手順）

ここでは“まず切る → 断面を整える → フランジを足す”の順にします。

### 1) 分割線/分割面を作る

- 分割線をカーブで描くなら: `InterpCrv` / `ControlPointCurve`
- 分割面を作るなら:
  - `ExtrudeCrv`（分割線を押し出して“切断面”を作る）
  - または `PlaneThroughPt`（単純な平面分割なら）

> スクショ位置案: 分割線→押し出し面（ExtrudeCrv）の作成

### 2) 本体を切る

- Brep（NURBS）なら: `Split`（面で分割）または `BooleanSplit`
- メッシュなら: `MeshSplit`（成功率が不安定なので、可能ならSubD/Brepに寄せる）

### 3) 合わせ面（フランジ）を足す

“切った断面”からフランジを作るのが安定です。

1. 断面エッジを取り出す: `DupEdge`（必要なエッジを複製）
2. フランジ幅を作る: `Offset`（曲線を外側/内側へオフセット）
3. フランジ面を作る: `Loft`（2本の曲線から帯状面）または `PlanarSrf`（平面なら）
4. 厚みが必要なら: `ExtrudeSrf`（フランジを押し出す）
5. 最後に `Join`

> スクショ位置案: DupEdge → Offset → Loft でフランジ帯を作る

## 首の大開口：空洞化と強度の両立

首開口は構造上の弱点になりやすいです。

- 開口縁に **厚みと幅を確保**（縁が欠ける/割れるのを防ぐ）
- 必要なら **リング状の補強**や、内側に **リブ**を入れる
- 分割と首開口が近い場合は、フランジと補強が干渉しないように先に設計する

### Rhinoコマンド例（首開口）

- 開口の輪郭を作る: `Circle` / `Ellipse` / `InterpCrv`
- 切り欠き形状を作る: `ExtrudeCrv`（カッターを作る）
- 穴を開ける: `BooleanDifference`（Brep同士で差分）

## 目などの後加工穴：基準を持って“後で確定”する

「目は後で穴を開ける」場合に重要なのは、穴そのものより **穴の基準**です。

### 基準の作り方（おすすめ）

- **基準平面**: 正中面、目の高さ面、顔の前後基準面
- **基準点**: 目頭/目尻、瞳中心、鼻先、口角など
- **基準線**: 正中ライン、目のライン（左右）

穴あけは、「基準 → 寸法（半径/楕円） → 位置決め」の順にすると、変更が入っても追従しやすいです。

### Rhinoコマンド例（目の穴を後で確定する）

1. 目の基準点（瞳中心）を置く: `Point`（レイヤーで管理）
2. 穴の輪郭を作る: `Circle` / `Ellipse`
3. “穴用カッター”を作る: `ExtrudeCrv`（貫通する長さ）
4. 穴を開ける: `BooleanDifference`

> スクショ位置案: 基準点→輪郭→カッター→BooleanDifference

### 穴の縁（エッジ）を“設計要素”にする

穴は開けた瞬間に局所応力が集中します。FDMでは特に割れやすいので、穴縁周りに **厚み**や **当たり面**を前提として残す設計が安全です。

## チェックリスト（分割/開口/穴）

- [ ] 分割面が組付けに耐える（手が入る、ネジが回る、位置決めできる）
- [ ] フランジを作った（平面/一定断面、位置決め要素を検討）
- [ ] 首開口縁に補強を入れる前提がある（割れやすい箇所）
- [ ] 目などの穴は“基準”で管理している（後で位置変更できる）

# エクスポート（STL/3MF）と最終検証（スライサーで潰す）

この章は、Rhino側で「それっぽく見える」から **実際にプリントできる**へ持っていくための最終チェックです。

※ FDMはスライサー設定で結果が大きく変わるので、Rhino単体で完結させず **必ずスライサーで検証**します。

## 出力形式の選び方

- **STL**: 互換性が高い（どのスライサーでもOK）
- **3MF**: 単位やメタ情報が保持されやすい（環境が揃うなら便利）

まずはSTLで通し、運用が固まったら3MFを検討、が安全です。

> 他CAD/解析/CAMへ渡す場合の注意（STEP/IGES含む）: `subd-cad-interoperability-and-formats.md`

### Rhinoコマンド例（出力）

- `Export`（選択書き出しなら `ExportSelected`）
  - STL / 3MF を選択
  - 単位やメッシュ密度（粗すぎない）を確認

## Rhino側の最終チェック（最低限）

- **ソリッド（閉じ）になっているか**
  - 穴/非多様体が残ると、スライサーで壁が崩れる
- **薄肉が意図通りか**
  - 1mmベースでも、強曲率で局所的に薄くなっていることがある
- **分割/開口/穴の縁が弱くないか**
  - 首開口、目の穴、合わせ面の端は割れやすい

Rhinoコマンド例:

- `Check` / `SelBadObjects`（壊れているものを検出）
- `ShowEdges`（穴＝Naked edges）
- `ClippingPlane`（薄肉/干渉を断面で確認）

## スライサーでの検証項目（FDMの必須チェック）

- **壁数（Perimeters）**
  - “1mm”が壁何周になるかを確認する（ノズル径依存）
- **薄肉検出**
  - 「壁が1周しか入らない/0になる」領域を見つける
- **サポート**
  - 前後分割でサポートが減っているか
- **強度の見積もり**
  - 首開口周り、コネクタ周り、合わせ面のフランジの強度

スライサーで問題が見つかったら、「形状を微修正」より先に **分割面・当たり面・補強の設計** を見直す方が戻りが少ないです。

## 失敗の典型と修正方針

- **穴が空いている/メッシュが壊れている**
  - 修正方針: その箇所を“設計として開口にする”か、殻として閉じ直す
- **薄肉が出る**
  - 修正方針: 局所補強（リブ/フランジ）を後段で追加する
- **合わせ面が歪む**
  - 修正方針: フランジ形状・位置決め（段差/キー）を見直す

## チェックリスト（出力前）

- [ ] `SelBadObjects` で問題が出ていない
- [ ] `ShowEdges` で穴（Naked edges）が残っていない（または意図した開口）
- [ ] 断面（`ClippingPlane`）で薄肉/干渉がないことを確認した
- [ ] スライサーで壁数・薄肉・サポート・強度見積りを見た

## テストピース（スライサー設定の最適化）

- **壁数テスト**: 壁2/3/4周で同形状を出して、強度/重量/時間を比較
- **ブリッジ/サポートテスト**: 分割面の向きがサポート量に与える影響を比較
- **嵌合クリアランス**: 0.1〜0.6mmの嵌合ラダーで“実際に動く”値を確定
## 次にやること

ここまでで、メッシュ原型から「厚み・分割・穴・コネクタを含む外装」を製造に持っていくための基礎が揃いました。

次はプロジェクト固有の要件（プリンタ/材料/機構）に合わせて、数値（壁数・隙間・補強寸法）をテストピースで固めていくのが最短です。

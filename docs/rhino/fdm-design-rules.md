# FDM向け外装設計ルール（Rhino 8 / アニマトロニクス）

この章は「顔の外装をFDMで出す」前提で、**後工程（穴あけ・機構CAD追加・分割・補強）に耐える**設計ルールをまとめます。

## まず決める：この外装の“基準”はどれ？

「1mm厚」を作る前に、厚みの基準を決めます。

- **外形優先（おすすめ）**: 外観のシルエットを守りたい。内側へ厚みを付ける。\n  - 長所: 見た目が崩れにくい\n  - 短所: 内部機構との干渉調整が必要になりやすい
- **内側優先**: 内部機構の空間を守りたい。外側へ厚みを付ける。\n  - 長所: 機構側のクリアランス管理がしやすい\n  - 短所: 外観が膨らみやすい
- **中央値優先**: 外側/内側に半分ずつ。\n  - 長所: 変化が平均化される\n  - 短所: どちらも少しずつズレる

## 肉厚の基本（FDM前提）

「1mm」は**薄肉**です。失敗率を下げるために、数値を壁数に変換して考えます。

- **壁厚（Wall）**: ノズル径 0.4mm の場合、壁2〜3周で 0.8〜1.2mm が現実的\n  - 1.0mm狙いなら「壁2周＋少し余裕」や「壁3周」を想定
- **局所補強は“あとで足す”**: まず均一厚みで閉じた外装を作り、最後にボス/リブ/フランジを足す\n  - 最初から可変厚みを狙うと、自己交差や穴あきが起きやすい

## クリアランス（機構との隙間）

機構CADを後で入れるなら、外装は「後から削る」のではなく、**最初から隙間を確保**します。

- **嵌合しない“逃げ”**: 可動・配線・組付けを想定して、局所的に余裕を持たせる\n+  - 標準値はプリンタ/材料で変わるので、最終的にはテストピースで決める
- **基準の持ち方**: 目・鼻・口などのランドマークは「基準平面/基準点」で管理して、穴位置を後から確定できるようにする

## 分割（前後2分割）設計のコツ

分割面は見た目だけでなく、**プリント性**で決めます。

- **サポートが減る向き**: できるだけ自然に置ける分割にする\n+  - 顔の前面はサポートだらけになりがちなので、分割面で逃げる
- **合わせ面は“平ら”を確保**: 有機形状のまま合わせると、後でズレやすい\n+  - 合わせ面（フランジ）を設け、ねじ/嵌合/マグネットなどの機構を載せる
- **強度の弱点を補強**: 首の大開口は特に割れやすい\n+  - 周辺にリブ/フランジを足す設計を前提にする

## よくある失敗（先に避ける）

- **薄肉＋強曲率で自己交差**: 厚み付けが破綻 → 形状を分割して局所ごとに処理\n- **合わせ面が曲面のまま**: 組付けでズレる/隙間が出る → フランジで“面”を作る\n- **穴あけを先にやりすぎる**: 後で位置が変わる → 基準だけ作って、穴は後段で確定\n

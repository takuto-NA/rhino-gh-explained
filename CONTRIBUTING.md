# Contributing

## セットアップ

- Node.js: **20以上**（推奨: 20.x）

```bash
npm install
npm run docs:dev
```

## ページ追加（推奨ルール）

このリポジトリは **サイドバーを自動生成**します。基本的に **`docs/` 配下に Markdown を追加するだけ**でOKです。

- Rhino: `docs/rhino/*.md`
- Grasshopper: `docs/grasshopper/*.md`

### 並び順を制御したい場合

Markdown先頭に `order` を入れてください（小さいほど上に表示）。

```md
---
order: 10
---

# タイトル
```

### タイトル

サイドバーの表示名は、原則として Markdown 内の最初の `# 見出し` を使います。

## 執筆方針（重要）

文章のルール（出典明記、ファクトチェック、口語の抑制、アルゴリズム/数式/再現手順による裏付け）は、次を参照してください。

- **正**: `docs/writing-guidelines.md`（サイト内で表示されるガイド）
- 補助: `WRITING_GUIDE.md`（GitHubで要点だけ見たい場合のショート版）

### コミット前チェック（要点）

- [ ] 断定している記述に根拠（出典/再現手順/数式）がある
- [ ] 推測は断定していない（必要なら推測と明示し、確認方法がある）
- [ ] 口語・感想語が過多になっていない
